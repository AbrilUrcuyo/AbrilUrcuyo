## 🚀 Featured Projects

### 🔹 StoreIT — Web & Desktop File Sync

**What it does:**  
A dual web and desktop system for secure file storage and synchronization.  
The web application (PHP + MariaDB) provides user and role management, multi-factor authentication (2FA), IP audit logs, cloud file browsing, storage quotas, and account controls.  
The desktop client (Python + SQLite) synchronizes a selected local folder with the server, supporting pause/resume, per-user sessions, device/IP validation, and local activity/status monitoring.

**Tech:** PHP · MariaDB · Python · SQLite  

**Status:** Currently private. Scheduled for open-source release in June 2026.

### 🔹 Distributed Message Broker (Mini-Kafka)

[🔗 GitHub Repository](https://github.com/AbrilUrcuyo/Message_Broker)

**What it does:**  
A distributed messaging system inspired by Kafka, implemented in C.  
Supports producers, a central broker, and consumer groups with round-robin load balancing.  
Includes message persistence, synchronization using mutexes and semaphores, and efficient handling of concurrent connections.

**Tech:** C · TCP sockets · Threads · Mutexes · Semaphores
